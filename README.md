# VSHacks2026-DISScord
DISScord, the newest Discord server protection system. Featuring a network of bots and a series of defenses, these bots will keep your server safe and your intruders emotionally damaged.

This is a project for VSHacks2026. Please note that there is a file for each of the bots. Some bots are very very similar, others not so much, so take care in finding the correct ones. 
There are 3 types of bots, an authenticator bot, aggravator bots, and lackey bots. The authenticator code is of type authenticator; the aggravator and verificator code are of type aggravator; and the L, Kira, Ryuk, Mafia, Mafia2, and Mafia3 code are of type Lackey.
There are also wildcards such as rapper, RapperFriend, and ChatBot. These bots bounce off of each other or act as basic AI assistants.
